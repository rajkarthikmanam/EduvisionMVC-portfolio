# 🎓 EduVision MVC

**EduVisionMVC** is a dynamic academic management web application built with **ASP.NET Core MVC** and **Entity Framework Core**, designed to demonstrate full-stack capabilities — from CRUD operations to interactive visualizations and intelligent chat integration.

---

## 🚀 Features

- 🔐 **Role-based user access** *(Admin, Instructor, Student)*  
- 📊 **Interactive dashboards** using Chart.js  
- 🧠 **AI-powered chatbot** integration via **Botpress**  
- 🗂️ **Complete CRUD** for Students, Courses, Departments, and Enrollments  
- 🌐 **RESTful API layer** for data access  
- 🏗️ **Modular architecture** using EF Core migrations and seed data  
- 💬 **Modern UI** (Bootstrap 5 + custom theme)  

---

## 🧩 Tech Stack

| Layer | Technology |
|--------|-------------|
| **Frontend** | Razor Views, HTML5, CSS3, Bootstrap 5 |
| **Backend** | ASP.NET Core MVC (.NET 9), C#, Entity Framework Core |
| **Database** | SQLite (Local), ready for SQL Server/Azure |
| **Data Viz** | Chart.js |
| **AI Chat** | Botpress Cloud Webchat v3 |
| **Version Control** | Git + GitHub |
| **Deployment** | Azure App Service / IIS |
